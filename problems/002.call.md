### 描述

Function 实例的 call() 方法会以给定的 this 和参数调用该函数。

请实现这份方法，命名为 myCall。

### 代码

```js
// 实现
Function.prototype.myCall = function () {};

// 测试
function sum(a, b) {
  return this.value + a + b;
}

const obj = {
  value: 3
};

// 输出: 6
console.log(sum.myCall(obj, 1, 2));
```

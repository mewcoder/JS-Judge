### 描述

编写一个 deepMerge 函数，该函数能够深度合并两个对象的属性。

如果两个对象中存在相同的属性，且属性值都是对象，则应该递归地合并这些属性对象；

如果属性值不是对象，则后面对象的值应覆盖前面对象的值。数组和函数应被视为不可合并的值，直接进行覆盖。

### 代码

```js
function deepMerge(target, source) {}

// 测试
const target = {
  a: 1,
  b: {
    c: 2,
    d: { e: 3 }
  },
  f: [4, 5],
  g: function () {}
};

const source = {
  a: 'one',
  b: {
    c: 'two',
    d: { e: 'three' },
    h: { i: 'four' }
  },
  f: [6],
  g: function sayHello() {}
};

console.log(deepMerge(target, source));
```
